======= SUPER MEME EDITION FU EDITION =======

Super Meme Fighter Fu Edition is a fighting game set in the lore of Internet Meme.

The game allow modification in the sense that any character can be added. See Existing character for all information.

=== SRC NOTES ===

The compilation need dirent.h and SFML 2.0 beta. Due to the disparition of the SFML version used, the precompiled WIN32 
binaries are included in the ext directory. The Visual Studio Solution is pre-set to compile with this directories.
